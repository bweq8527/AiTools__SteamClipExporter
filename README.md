# SteamClipExporter

**项目简介**

​	steam自带的视频录制功能非常便利，但无法批量导出为常用格式的视频文件，因此借助AI开发了此工具。

**使用方法**

<方法1>

1. 运行服务器：
   双击根目录下的 run_server.bat 文件

2. 访问前端界面：
   服务器启动后，在浏览器中访问： http://127.0.0.1:5000/

<方法2>

​	双击main.py，在IDE环境（如Pycharm）下运行。

<方法3>

​	在IDE环境下运行build.py，等待脚本自动打包结束后在dist文件夹下获取EXE文件。

> 方法2、3需配置环境并安装相应依赖项FFmpeg (必须下载并将其 bin 目录添加到系统环境变量 Path 中)

**操作步骤**

1. 在界面上方的输入框中，输入 Steam 录像根目录（例如：H:\Steam\clips）。
2. 点击“设置路径并扫描”。
3. 在左侧列表中多选想转换的录像（选择顺序即为转换顺序）。
4. 点击“加入转换队列”。
5. 右侧将实时显示转换进度和柱状图。
